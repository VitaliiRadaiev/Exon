Exon.Планер

Модуль календарно-сетевого планирования, мониторинга и ведения строительных задач

Заказать презентацию

Теперь
вы можете

  • Создавать/редактировать/удалять задачи.
  • Cледить за выполнением работ.
  • Контролировать сроки сдачи.
  • Задавать необходимые для работ показатели: даты, объемы, стоимость.
  • Выстраивать структуру и последовательность работ.
  • Структурировать все виды работ.

Модуль
автоматически

  • Разделяет уровень доступа между участниками проекта.
  • Собирает информацию с других модулей.
  • Показывает фактические показатели выполнения работ.
  • Отображает график выполнения работ.
  • Пересчитывает показатели работ при корректировках сроков.
  • Выгружает отчет по суточно-месячному выполнению работ.

Решения и возможности

  • Ведение графика в WBS-структуре
  • Построение диаграмм Ганта
  • Импорт графиков в Excel, MS Project
  • Экспорт графиков в pdf, xlsx, mpp
  • Отображение истории изменений
  • Контроль объемных и cтоимостных показателей
  • Отслеживание прогресса работ
